## Installation
Fork this repository or download
with: ```git clone https://github.com/Knutakir/emorjis```

Then run ```npm start```.

Or if you dont want
an express server for the game, then navigate to the ***src***
folder and run [index.html](src/index.html) in your webbrowser.
